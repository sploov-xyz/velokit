### **What is VeloKit?**
VeloKit is a high-velocity CLI engine designed to scaffold production-ready applications in seconds. Originally built for Discord bots, v1.0.1 expands to support generic APIs and multiple languages.

### **Features (v1.0.1)**
- **Multi-Project Support:**
  - ü§ñ **Discord Bots:** Professional handler architecture with modular "Souls" (Music, AI, Moderation).
  - üöÄ **Express APIs:** Instant REST API scaffolding with optional MongoDB/Docker support.
- **Dual-Language:** Native support for **TypeScript** and **JavaScript (ESM)**.
- **The Build Process:** A 7-phase interactive TUI that customizes every aspect of your project.
- **Modular Engines:**
  - **Music:** Lavalink / NodeLink / Riffy integration.
  - **AI:** Google Gemini / Groq / OpenAI adapters.
  - **Infrastructure:** Docker Compose & Mongoose/Prisma generators.

### **Quick Start**
No installation required. Just run:

```bash
npx @sploov/velokit
```
