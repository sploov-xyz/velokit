# Contributing to VeloKit

First off, thank you for considering contributing to VeloKit! It's people like you who make VeloKit such a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the issues to see if it has already been reported.
* If not, open a new issue with clear steps to reproduce the bug.

### Suggesting Enhancements
* Open an issue describing the feature and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## Project Structure
- `src/index.ts`: The main CLI logic.
- `src/templates/`: The bot architecture templates.
- `scripts/`: Build and distribution scripts.
